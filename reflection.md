# Reflection: Compare Two Integers Program

## 1. What the program does
Reads two integers from user input, compares them using `if-else if-else`, and prints which one is greater or if they’re equal.

## 2. Key concepts used
- **cin >>**: For reading multiple inputs in one line
- **if-else if-else ladder**: To handle 3 mutually exclusive cases: `>`, `<`, `==`
- **Relational operators**: `>`, `<`, `==` to compare values
- **Decision making**: Program chooses one output path based on condition

## 3. What I learned
1. `else if` is cleaner than writing multiple separate `if` statements when cases are related.
2. Order matters in `if-else
